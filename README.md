## 📚 Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/techieRahul17/googleAihack.git
   ```
2. **Install dependencies:**
   ```sh
   pip install -r requirements.txt
   ```
3. **Explore a project:**
   ```sh
   cd Visionary
   jupyter notebook
   ```
